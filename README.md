# Note_Taker

https://rubenstein-note-taker.herokuapp.com/

## Application

This app is designed for the user to create a list of notes.  The user also has the ability to delete notes.  The app will number each note and update depending on the user's deletions and insertions.

### Tools

- Jquery

- HTML

- Bootstrap

- Express

- MySQL

- Heroku
